Dynamic Routing for Marine Debris Collection

This project explores a reactive strategy to maximize plastic collection by routing autonomous vessels through drifting hotspots.

  • Hotspots are modeled as dynamic high-density points
  • Each has a value and moves with ocean currents
  • We combine value and proximity using a score function

Mathematical Model

The goal is to maximize collected value over a time horizon using a weighted score for each hotspot:

\\[ \\text{Score}_i = \\alpha \\cdot \\left(1 - \\frac{d_i}{d_{\\text{max}}}\\right) + (1 - \\alpha) \\cdot \\left(\\frac{v_i}{v_{\\text{max}}}\\right) \\]

  • \\(d_i\\): distance to hotspot \\(i\\)
  • \\(v_i\\): value of hotspot \\(i\\)
  • \\(\\alpha\\): trade-off parameter (\\(0 \\leq \\alpha \\leq 1\\))

Greedy strategy with Œ±:

  1. Evaluate scores for all reachable hotspots
  2. Select the one with the highest score
  3. Move toward it and collect
  4. Re-evaluate and repeat
